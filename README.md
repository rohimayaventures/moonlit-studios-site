# Moonlit Studios Site

A Next.js + Tailwind CSS portfolio for Moonlit Studios.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

Pages included:

- `/` — Home (hero, snapshot of services)
- `/services` — Service suites (Brand, Web, AI, Author)
- `/portfolio` — Placeholder case studies
- `/about` — Studio overview
- `/contact` — Static contact form

This is a starter you can freely customize and extend.
